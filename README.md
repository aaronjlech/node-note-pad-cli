# node-note-pad-cli

## Quick little node app that writes/reads/removes notes to a JSON using the CLI. 


## Commands

### Add Note
`node app.js add --title="$titleName" --body="$bodyText"`

### Remove Note
`node app.js remove --title="$titleName"`

### Read Single Note
`node app.js read --title="$titleName"`

### List all Notes
`node app.js list`
